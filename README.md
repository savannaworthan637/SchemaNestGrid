# SchemaNestGrid - Nested Structured Data Generator

SchemaNestGrid allows developers to define field schemas in grid formations for realistic and complex data generation.

## Features
- Multi-layered nested schemas  
- CSV / JSON / SQL export  
- CLI-friendly
